# DevOps Engineer — Health Data Pipeline Infrastructure

- **Period:** 01/2021 – 05/2022
- **Role:** DevOps Engineer
- **Client:** MediFlow B.V.
- **Intermediary:** TechStaff Solutions
- **Industry:** Healthcare / Health-Tech
- **Location:** Netherlands (NL)
- **Type:** contract

## Description

Designed and operated the infrastructure for a health data analytics platform processing anonymised patient data from 200+ clinics. GDPR compliance and data isolation were critical requirements. The platform ingested, transformed, and served analytics dashboards for hospital administrators.

## Responsibilities

- Designed AWS infrastructure for data pipeline (ECS, Step Functions, S3, Redshift)
- Implemented infrastructure as code with Terraform across 4 AWS accounts
- Built CI/CD pipelines in GitLab CI for 8 Python data processing services
- Configured network isolation and encryption for GDPR compliance
- Set up centralized logging (ELK stack) and monitoring (CloudWatch, Grafana)
- Automated database backup and disaster recovery procedures
- Participated in GDPR audit preparation — documented all data flows and access controls

## Key Achievements

- Passed GDPR audit with zero findings on infrastructure controls
- Reduced data pipeline processing time by 40% through infrastructure optimisation
- Automated previously manual deployment process — from 2 hours to 15 minutes
- Designed cross-account networking that became the template for 3 subsequent projects

## Technologies

AWS (ECS, Step Functions, S3, Redshift, CloudWatch, IAM), Terraform, Docker, Python, GitLab CI, Elasticsearch, Kibana, Grafana, PostgreSQL
